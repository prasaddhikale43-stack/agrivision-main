**AgriVision: AI-Powered Agricultural Intelligence**
AgriVision is an advanced web platform designed to bridge the gap between farmers and technology. By utilizing Generative AI (Gemini), Computer Vision, and real-time data analytics, the platform offers a comprehensive suite of tools for crop diagnostics, predictive analytics, and expert consultation.

**Website Workflow**
The platform follows a structured user journey:
**Entry:** User starts through a web browser with personalized settings for Language, Theme, and Crop preferences.
**Core Pillars:** The site branches into four main service categories: Analytics, Diagnostics, Assistance, and Information.
**Intelligence Layer:** Powered by a React.js Frontend and a Node.js Backend, integrating Gemini AI for core processing.
**Use-Case Logic**
**Farmer Interaction:** Farmers interact with the website to initiate diagnostics and view history.
**Expert Integration:** Verified Agricultural Experts can manage data via the Firebase Console and provide direct human assistance.
**Data Flow:** Predictive analytics models handle crop disease and soil risk, storing results in a SQL Database and JSON Data Stores.
